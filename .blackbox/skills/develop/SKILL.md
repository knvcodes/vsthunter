## Rules

1. **Always write tests after coding** — Keep working until all tests pass.
2. **Optimize all images and videos** — Compress and format assets for performance.
3. **Remove unused code** — Clean up dead code, imports, and dependencies.
4. **Maintain existing code structure** — Follow the current patterns and conventions.
5. **Avoid `any` in code** — Use `unknown` instead of `any` for type safety.
6. **Always build the code** — Run the build step to confirm everything compiles and works correctly.
